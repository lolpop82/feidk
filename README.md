webpage that allows for solo drafting of fire emblem games if you want to just do challenge runs by yourself. 

uses AI for help w backend because hobby project

going to be hosted but to self host put all files in folder & open index.html 

features 
  - banned unit system
  - linear/snake draft
  - config round #
  - config bot #
  - config rng weights up to 999
  - config free units
  - light/dark mode
  - copy results
  - images for every character/game
    
games supported

fe5, fe6, fe7, fe8, fe9, fe11 ,fe12, fe13,fe14 (all routes),fe17
